# HEELKAWN WORLD BIBLE

This directory is the canonical world and lore memory for HeelKawn.

If chat history is lost, this directory is the recovery source.

## Read Order

1. `../HEELKAWN_STATE.md`
2. `MASTER_INDEX.md`
3. `TIMELINE.md`
4. The specific domain file you are updating

## Canon Rules

- Canon facts must be deterministic and explainable.
- New lore must align with project design rules.
- If a fact conflicts with existing canon, record a proposal first in `CANON_CHANGELOG.md`.
- Avoid rewriting old entries; append revisions with date and reason.
