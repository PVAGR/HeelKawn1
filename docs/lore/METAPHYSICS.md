# HeelKawn Metaphysics

Status: Canon reference  
Scope: Ontology and myth-structure boundaries for implementation and narrative interpretation.

## Ontological Principles

### Asha and Druj

- Asha and Druj are currents, not gods.
- They are directional pressures within reality:
  - Asha: coherence, continuity, and lawful alignment.
  - Druj: distortion, fracture, and entropic drift.
- Systems may model their influence as tendencies, never as deity intervention.

### Life-Death-Veil-Pergatory Cycle

- Canon soul/identity cycle:
  1. Life
  2. Death
  3. Veil
  4. Pergatory
- This cycle frames meaning and continuity; it does not grant automatic resurrection rights.
- Any return or legacy effect must be mediated by recorded memory and lawful simulation systems.

## Canon Boundary: Myth-Cycle vs Base State

### Taured / DRUJ / Ark

- Taured/DRUJ/Ark is a Myth-Cycle and Parallel-Expression track.
- It is not the starting state of baseline HeelKawn simulation.
- Base simulation must not assume Taured/DRUJ/Ark as active canonical present unless explicitly promoted by canon governance.

## Implementation Guardrails

- Metaphysical language may shape interpretation, UI tone, and narrative framing.
- It must not bypass deterministic kernel rules.
- Mythic constructs cannot overwrite recorded facts.
