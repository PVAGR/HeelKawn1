## 🔧 **TROUBLESHOOTING:**

### **UI Not Showing:**
1. Make sure HeelKawnUIAuto node is child of Main
2. Check Output panel for errors
3. Press H - help should show

### **Buttons Not Working:**
1. Make sure game is running (F5)
2. Check if buttons are visible
3. Try keyboard shortcuts (I, C, K, H)

### **Camera Not Moving:**
1. Right-click and hold while dragging
2. Try mouse wheel for zoom
3. Check if camera exists in scene

### **Inventory Empty:**
1. Right-click a tree/rock/bush
2. Click "Gather" option
3. Press I to see items
