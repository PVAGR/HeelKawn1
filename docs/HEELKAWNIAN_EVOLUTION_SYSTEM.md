# 🧬 HeelKawnian Indefinite Evolution System

**Created:** May 7, 2026  
**Vision:** Build AI that lets HeelKawnians evolve from primitive age to real-world technological standards over indefinite simulation time

**Core Insight:** Humans may think they are players, but they are meant to **roam the world and make it liveable**. The world lasts indefinitely — thus the AI must develop indefinitely.

---

## 🎯 THE VISION

### What Makes This Unique

HeelKawn is **not** a game with an end state. It is a **living simulation** where:

1. **HeelKawnians are the true protagonists** — not the player
2. **The world evolves indefinitely** — no cap on technological progress
3. **Civilization emerges from behavior** — not scripted events
4. **History is deterministic** — same conditions produce same outcomes
5. **Knowledge can be lost AND rediscovered** — like real history

### The Goal

> Build a world where pawns evolve from **stone-age survivors** to **technological civilization** over hundreds of generations, driven entirely by their own AI, decisions, and accumulated knowledge.

**Player role:** Observer, not controller. The Truman Show director who watches the story unfold.

## 🏗️ ARCHITECTURE FOR INDEFINITE EVOLUTION

### Layer 1: Civilizational Stages

```
Stage 0: Primitive     → Hunter-gatherer, basic tools, fire
Stage 1: Neolithic     → Farming, pottery, permanent settlements
Stage 2: Bronze Age    → Metalworking, writing, cities, trade
Stage 3: Iron Age      → Iron tools, empires, roads, currency
Stage 4: Medieval      → Universities, guilds, cathedrals, printing
Stage 5: Renaissance   → Science, exploration, art, banking
Stage 6: Industrial    → Steam, factories, railroads, mass production
Stage 7: Modern        → Electricity, combustion engine, telecommunications
Stage 8: Information   → Computers, internet, digital knowledge, AI
Stage 9: Space Age     → Off-world colonies, fusion, genetic engineering
Stage 10+: Post-Scarcity → Replicators, immortality, god-tech
```

**Key Design:** Stages are **emergent**, not scripted. Civilization reaches Stage N when:
- Required technologies researched
- Required knowledge types widespread
- Required infrastructure built
- Required social complexity achieved

### Layer 2: Innovation System

**Current:** Technology tree is predefined (pawns unlock what we defined)

**Needed:** Pawns can INVENT new technologies by:

1. **Combination**
   - Knowledge A + Knowledge B = New Knowledge C
   - Example: Fire Making + Clay Working = Pottery (discovered, not unlocked)

2. **Experimentation**
   - Pawn tries new approach to old problem
   - Success rate based on Knowledge, Intelligence, Tools
   - Failure can produce disasters (fire, injury) or nothing

3. **Accident**
   - Random discovery during unrelated activity
   - Lightning strikes tree → discovers metal smelting
   - Rate based on activity level, material diversity

4. **Cross-Pollination**
   - Traders bring foreign knowledge
   - Refugees bring lost techniques
   - War spoils include captured texts/artifacts

### Layer 3: Knowledge Preservation Evolution

**Current:** Knowledge dies with last carrier, or is inscribed on stones

**Needed:** Evolving preservation methods:

| Era | Storage Method | Capacity | Longevity | Searchability |
|-----|---------------|----------|-----------|---------------|
| Primitive | Oral tradition | ~10 stories | 3-5 generations | Poor (memory-dependent) |
| Neolithic | Carved stones | ~100 facts | 100+ years | Poor (must visit stone) |
| Bronze Age | Clay tablets | ~1000 facts | 500+ years | Medium (indexed by scribes) |
| Iron Age | Papyrus scrolls | ~10,000 facts | 200+ years | Medium (library catalogs) |
| Medieval | Paper books | ~100,000 facts | 1000+ years | Good (dewey decimal) |
| Industrial | Printed books | ~1M facts | 500+ years | Excellent (card catalogs) |
| Modern | Digital storage | ~1B facts | 50+ years (decay) | Perfect (search engines) |
| Information | Cloud databases | ~1T facts | Indefinite (backups) | Perfect (full-text search) |
| Post-Scarcity | Crystalline storage | ~1P facts | Million years | Instant (neural link) |

**Implementation:** Each storage method is a building that can be constructed when prerequisites met.

### Layer 4: Research Institutions

**Current:** Individual pawns research alone

**Needed:** Collective research systems:

1. **Apprenticeship** (Neolithic)
   - Master teaches 1-3 apprentices
   - Knowledge transfer rate: +50%
   - Knowledge loss risk: -30%

2. **Guilds** (Bronze Age)
   - Profession-wide knowledge sharing
   - Research collaboration
   - Standards enforcement

3. **Universities** (Medieval)
   - Dedicated research buildings
   - Multiple researchers working together
   - Knowledge preservation (library)
   - Teaching programs

4. **Research Labs** (Industrial)
   - Specialized by field (chemistry, physics, biology)
   - Equipment bonuses
   - Team research projects

5. **Think Tanks** (Information)
   - Virtual collaboration
   - AI-assisted research
   - Global knowledge access

### Layer 5: Complexity Metrics

Track civilization sophistication:

**Economic Complexity:**
- Specialization index (how many distinct professions?)
- Trade network size (how many settlements trade together?)
- Currency sophistication (barter → coin → credit → crypto)

**Social Complexity:**
- Governance layers (tribe → chiefdom → state → empire → federation)
- Law sophistication (oral tradition → written code → constitution → international law)
- Social safety nets (none → family support → guild support → state welfare)

**Technological Complexity:**
- Energy capture (human muscle → animal → water/wind → steam → electric → nuclear → fusion)
- Material sophistication (stone → bronze → iron → steel → plastic → composites → nanomaterials)
- Information processing (oral → written → printed → digital → quantum)

**Quality of Life:**
- Lifespan (average age at death)
- Literacy rate (% who can read)
- Health (disease prevalence, injury survival)
- Leisure time (hours/day not spent on survival)
- Security (violent death rate)

## 🔧 TECHNICAL DESIGN

### Civilization Stage Calculation

```gdscript
func get_civilization_stage(settlement_id: int) -> int:
    var score: int = 0
    
    # Technology score (0-30 points)
    var techs = TechnologySystem.get_researched_technologies(settlement_id)
    score += min(30, techs.size() * 2)
    
    # Knowledge score (0-20 points)
    var knowledge_types = KnowledgeSystem.get_known_types(settlement_id)
    score += min(20, knowledge_types.size())
    
    # Infrastructure score (0-20 points)
    var buildings = SettlementMemory.get_buildings(settlement_id)
    score += min(20, buildings.size() / 5)
    
    # Complexity score (0-20 points)
    var professions = get_profession_diversity(settlement_id)
    score += min(20, professions * 3)
    
    # Quality of life score (0-10 points)
    var lifespan = get_average_lifespan(settlement_id)
    score += min(10, int(lifespan / 10))
    
    # Convert score to stage
    if score < 20: return 0  # Primitive
    if score < 40: return 1  # Neolithic
    if score < 60: return 2  # Bronze Age
    if score < 80: return 3  # Iron Age
    if score < 100: return 4  # Medieval
    if score < 120: return 5  # Renaissance
    if score < 140: return 6  # Industrial
    if score < 160: return 7  # Modern
    if score < 180: return 8  # Information
    if score < 200: return 9  # Space Age
    return 10  # Post-Scarcity
```

### Innovation Combination System

```gdscript
func try_innovate(pawn_id: int, knowledge_a: int, knowledge_b: int) -> bool:
    var pawn = get_pawn(pawn_id)
    var rng = WorldRNG.for_pawn(pawn_id, GameManager.tick_count)
    
    # Base success rate
    var base_chance: float = 0.05  # 5%
    
    # Bonuses
    base_chance += pawn.intelligence * 0.01
    base_chance += get_knowledge_level(pawn_id, knowledge_a) * 0.02
    base_chance += get_knowledge_level(pawn_id, knowledge_b) * 0.02
    
    # Tool bonus
    base_chance += get_tool_bonus(pawn_id) * 0.03
    
    # Institution bonus (university, guild)
    base_chance += get_institution_bonus(pawn_id) * 0.05
    
    # Check for success
    if rng.float() < base_chance:
        # Success! Create new knowledge
        var new_knowledge = _combine_knowledge(knowledge_a, knowledge_b)
        _record_discovery(pawn_id, new_knowledge)
        return true
    
    return false
```
